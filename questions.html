Problem Statement 1:

Download as 
Write a program to simulate this situation.

Input sequence:
First line of the input will have the name of the two teams that are going to compete. Second line has two numbers n m.
Where n represents the number of supporters and m give the number of reporters.
Second line is followed by n lines where each line represents a supporter.
First word of the line is the name of the supporter and the second one tell the team that she/he supports.
This is followed by m line where each line represents a reporter.
First word is the name of the reporter and second part tell the news channel she/he belongs to.

After the above input any number of inputs can be given. Each input will represent a goal.
It will be name of the team that had scored the goal.
Output to every goal will be response of each supporter and reporter. The final input will be "game over".
At this point the program will terminate and display the final response of all supporters and reporters.

Input :
i/p: India Pakistan
i/p: 2 1
i/p: Ravi, India
i/p: Rahul, Pakistan
i/p: Vijay, CNN news
i/p: India
o/p: Ravi says: Hurrah!
o/p: Rahul says: Alas!
o/p: Vijay reports: India has scored a goal at 9:30am - Brought to you by CNN news.
i/p: game over.
o/p: Ravi say Yee! India won.
o/p: Rahul says Alas! Pakistan lost.
o/p: Vijay reports:India has won game against Pakistan by 1-0 - Brought to you by CNN news.
