About

ThoughtWorks Hyderabad is glad to announce a free-for-all Xtreme Mobile for developers workshop. This is a series of events designed to build a community of mobile developers. Each editon will comprise of a different theme based on the level of expertise targetted, practices involved, platform used and so on.


What's in this Edition

It is a 3-day hands-on workshop that aims to teach the basics of mobile app developmenet using Android.

Topics that will be covered

  • Day 1
  • Android Architecture
  • Android Environment
  • Activity Lifecycle
  • Designing User Interface using activities and Layouts
  • Day 2
  • Fragment Lifecycle
  • Building the User Interface for different Devices using Fragments
  • Data storage
  • Security considerations
  • Day 3
  • Handling REST calls and async tasks
  • Interaction with other Apps (Eg: Contacts)
  • Introduction to testing frameworks
  • Publish app to Play store

Who can apply?

Should have experience in Java to follow the workshop.

It is NOT restricted to only Hyderabadis. If you can manage your own travel and stay, you are most welcome to apply.

What is the selection Criteria?

Selection will be based on assessment of your code for a given coding problem. There are two problems in the link provided below, you can solve any ONE of them only in Java.

The difficulty level of the problem is basic, mainly targetted to assess your coding style and understanding of Java.

Evaluation of your code will be based on various parameters like object-oriented design and good coding practices.
