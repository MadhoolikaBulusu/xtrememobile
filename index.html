About

ThoughtWorks Hyderabad is glad to announce a free for all Extreme Programming for developers camp. It is a 4-day course with coding and hands on exercises. Build the community of developers who practice extreme programming(XP) methodologies in software development life cycle. This course will help the developers to gain hands on experience of XP methodologies which can be applied to build the better softwares.

Topics that will be covered

  • Pair Programming
  • Unit Testing (Automated Testing)
  • Test Driven Development
  • Refactoring
  • Continuous Integration and deployment
  • Object oriented programming with some bits of Design Patterns
  • Along the way you will also learn Standups, Retrospectives and other Agile practices.

What is the selection Criteria?

Selection will be based on assessment of your code for a coding problem.

The coding problem will be sent via emails to all those who register. You can use any programming language of your liking to solve it. By design, the difficulty level of the problem will be low. It will test your basic coding skills and commitment to finish a given task (hopefully, implying the commitment to finish the course). Use your object oriented design and automated testing skills to solve the problem.

Who can apply?

You must be a hands on programmer to apply. It does not matter if you are a university student, an experienced polyglot programmer or a non-comp-science professional who knows programming, as long as you can take a simple coding test.

It is NOT restricted to only Hyderabadis. If you can manage your own travel and stay, you are most welcome to apply.

Why free?

Well, we have been active evangelists of these development practices for many years. Today we are very happy to see that there is increased interest and awareness of extreme programming concepts across India and in Hyderabad as well. However, we feel that there is a big gap in the understanding and appreciation of these practices. We would like to share our knowledge to build community around these practices.
